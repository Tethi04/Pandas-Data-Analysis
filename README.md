Overview

This project is a data analysis tool built for Task 5 of the Python Developer Internship. It uses the powerful Pandas library to process a raw CSV dataset of sales records and Matplotlib to visualize the results.
The script automatically calculates key metrics like total revenue and best-selling products, and it generates a professional bar chart image.
